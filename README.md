# git操作练习

1. 在github上创建仓库

2. 在本机执行

   `git remote add origin git@github.com:Voivio/demo.git `

   以添加到本地

3. 在本机执行

   `git push origin master`

   把本地内容推送到github上

4. 在github上对README.md进行修改，再在本地上运行

   `git pull origin master`
   
   把远端内容存回本地
   
5. 创建了testing分支，在本地merge之后把master和testing分支一并上传
